Scene Map ---------------

Units can now:
	Gather resources.
	Attack.
	Die(unstable, may cause crashes).


Directional arrows:
	Move the camera.

Numerical keys:
	1: Nexus tools.
		Spawn a probe.
	2: Probe tools.
		Move, Stop, Attack(not working),
		Build a Nexus, Build a Pylon.
	3: Build an assimilator (con only be built on vespene gas).
	4: Build a gateway.
	(Left click to place them when the marker is green)
	(Right click to cancel).

Letters:
	D: Enable debug mode.
	(Display mouse coordinates, walkability map, entities rects).
	C: UI debug.
	V: Display pathfinding nodes.


Mouse:
	Right click: move selected units.
	Mouse wheel: Create a Probe.
	Left click: Select a unit or a building.
		Hold to select multiple units at once.

Scene Gui ---------------
Numbers Pad:
	1 - 4: Decrease/Increase Minerals.
	2 - 5: Decrease/Increase Gas.
	3 - 6: Decrease/Increase Population.

Regular Keys:
	1: Change to Base's orders grid.
	2: Change to Probe's orders grid.

General   ---------------

Escape: Open the console.
Commands:
	list: display a list of commands.
	quit: close the application
